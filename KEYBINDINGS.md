# Keyboard Shortcuts

Complete reference for all keyboard shortcuts in ani-tui.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `q` | Quit application |
| `Shift+S` | Open search overlay |
| `Shift+C` | Toggle source selection |

## Splash Screen

| Shortcut | Action |
|----------|--------|
| `Enter` / `Esc` | Continue to source selection |

## Source Selection

| Shortcut | Action |
|----------|--------|
| `↑/↓` | Navigate sources |
| `Space` | Toggle selected source |
| `Enter` | Confirm and go to home |
| `Esc` | Go back |

## Home Screen

| Shortcut | Action |
|----------|--------|
| `Shift+S` | Open search |
| `q` | Quit |

## Search Overlay

| Shortcut | Action |
|----------|--------|
| `↑/↓` | Navigate search results |
| `Enter` | Select anime |
| `Shift+C` | Open source selection modal |
| `Esc` | Cancel search / Go back |
| `Type` | Search query |
| `Backspace` | Delete character |

## Source Selection Modal (in Search)

| Shortcut | Action |
|----------|--------|
| `↑/↓` | Navigate sources |
| `Space` | Toggle source |
| `Enter` | Confirm and update search |
| `Esc` | Close modal |

## Player

### During Playback

| Shortcut | Action |
|----------|--------|
| `Any key` | Show control menu |

### Control Menu Visible

| Shortcut | Action |
|----------|--------|
| `↑/↓` | Navigate controls |
| `Enter` | Select control |
| `n` | Next episode |
| `p` | Previous episode |
| `e` | Choose episode |
| `f` | Add to favorites |
| `b` | Back to menu |
| `Esc` | Hide controls |

### Episode List Modal

| Shortcut | Action |
|----------|--------|
| `↑/↓` | Scroll episodes |
| `Enter` | Play selected episode |
| `Esc` | Close modal |

### End Screen

| Shortcut | Action |
|----------|--------|
| `Enter` | Play next episode (if available) |
| `r` | Replay current episode |
| `b` / `Esc` | Back to menu |

## Tips

- At least one source must be enabled at all times
- Use `Shift+C` to quickly toggle between sources during search
- The control menu auto-hides after 5 seconds of inactivity
- On the end screen, `Enter` defaults to the most logical action
